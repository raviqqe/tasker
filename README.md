Offline-capable task management app based on [React](https://reactjs.org) and
[Firebase Firestore](https://firebase.google.com/docs/firestore/)

## Features

- Tasks per project
- Built-in pomodoro timer
- Data sync among devices
- Offline capability
- Mobile support

## Stack

- React & Redux
- TypeScript
- Firebase
  - Firestore
  - Authentication
